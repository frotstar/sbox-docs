# Effects

Visual effects you can add to a scene as components, including particles, beams, decals, and indirect lighting.

## [Particle Effect](/rendering/effects/particle-effect/index.md)

A CPU-simulated, fully programmable particle system built from components.

## [Beams](/rendering/effects/beams/index.md)

Animated line/laser effects, useful for things like tracers and lasers.

## [Decals](/rendering/effects/decals/index.md)

Project textures onto surfaces in the world.

## [Indirect Light Volumes](/rendering/effects/indirect-light-volumes.md)

Baked grids of probes that add bounced indirect lighting to an area.
